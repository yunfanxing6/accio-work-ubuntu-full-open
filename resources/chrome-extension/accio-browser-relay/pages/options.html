Quick Start
  1. Make sure Accio is running with browser control enabled.
  2. The relay connects automatically — a green dot on the icon means ready.
  3. Ask Accio a browser-related question — it can now see and control your tabs.
How It Works
Always-on relay
The relay is enabled by default and connects automatically when Accio is running. Use the toggle above to pause if needed. Agent-opened tabs are grouped under an "Accio Agent" tab group.
Auto-reconnect
If the connection drops, the extension reconnects automatically. No manual intervention needed.
Tab safety
Your existing tabs are protected — the agent can only close tabs it opened. Personal tabs are never affected.
Status Indicators
Connected — agent can control your tabs Connecting / reconnecting to local relay server Relay disabled — no indicator
Troubleshooting
Stays on yellow dot (connecting)
Relay server not reachable. Ensure Accio is running with browser control enabled.
Pages not responding
Internal Chrome pages (chrome://) cannot be controlled. Navigate to a regular webpage.
