Install Accio Browser Relay Extension

Install the extension to connect your browser to Accio.

Chrome Web Store Recommended

Install directly from the Chrome Web Store — the simplest way.

Install from Chrome Web Store
Or install manually from local files
  1. Open Chrome Extensions & Enable Developer Mode
    Then turn on the Developer mode toggle in the top-right corner of the page.
    Open a new Chrome tab and copy the address:
    chrome://extensions Click to copy
  2. Open the Extension Folder
    Open the folder that contains the browser extension files: chrome-extension/accio-browser-relay
  3. Drag the Extension Folder into Chrome
    Drag the accio-browser-relay folder directly onto the chrome://extensions page. Chrome will install it immediately.
    Load Unpacked
    Click "Load unpacked" on the chrome://extensions page and select the extension folder.
    Navigating to a path in Finder (macOS)
    In the file picker dialog, press ⌘ + Shift + G to open "Go to the folder", then paste the extension folder path and press Enter.
    Pin the Extension
    Click the puzzle piece icon in Chrome's toolbar and pin Accio Browser Relay for quick access.
    Updating the Extension
    When Accio is updated, go to chrome://extensions, remove the old version, then repeat the steps above. Or click the reload icon (↻) on the extension card to quick-reload.
    About the debugging banner
    When the extension attaches to a tab, Chrome shows a yellow banner: "Accio Browser Relay started debugging this browser". This is a standard Chrome security notice. Do not click "Cancel" unless you want to disconnect that tab.
Troubleshooting
Badge shows … — Accio is not running or browser control is not enabled.

Can't control a page — Internal pages (chrome://) cannot be debugged. Use a regular webpage.

Extension disappeared — Unpacked extensions may get disabled after Chrome restart. Re-enable at chrome://extensions.
