Choose Your Connection Method

Two ways to connect your browser. Choose the one that works best for you.

Recommended
Browser Extension

Install once, connect automatically. No repeated authorization needed.

  • Auto-connects on every launch
  • No manual approval required
  • Works with all Chrome versions
  • Requires installing a Chrome extension
CDP Direct Connect

Zero installation. Connect directly via Chrome DevTools Protocol.

  • No extension installation needed
  • Works out of the box on Chrome 144+
  • Requires re-authorization after every Accio or Chrome restart
  • Chrome 144+ only
